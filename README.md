# IPL-Auction-Analysis
Analysis of IPL Auction Data 2013-2023 using Python, SQL and Tableau

Objective
Analyzing 10 years of IPL auction data to uncover 
spending trends, player valuations and team strategies.

 Tools Used
- Python (Pandas, Jupyter Notebook)
- SQL (Microsoft SQL Server)
- Tableau Public (Dashboard)

 Files in this Repository
- `IPLauction.csv` - Cleaned dataset
- `Auction.ipynb` - Python cleaning notebook
- `IPLAuction.sql` - All SQL queries
- `Dashboard.png` - Tableau dashboard screenshot

 Key Questions Answered
1. Who are the top 10 highest paid players?
2. Which team spent the most overall?
3. How has auction spending changed over 10 years?
4. Which country produces most expensive players?
5. Indian vs Overseas player comparison
6. Which teams are most consistent?

 Key Findings
- Mumbai Indians, KKR, RCB and Sunrisers participated
  in all 11 auctions — most consistent teams
- Overseas players command higher average bids
- IPL auction spending grew significantly since 2013
- Punjab Kings got most players at lowest average cost
